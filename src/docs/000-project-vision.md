# Project Vision

This project explores the idea that a resume can be treated as a software project.

Instead of relying on a monolithic document or a visual template, the resume is designed with the same principles applied to software engineering:

- Modularity
- Maintainability
- Readability
- Simplicity
- Consistency

Every design decision should improve both the document itself and the experience of maintaining it over time.

The final PDF is the product, the source code is the project.
